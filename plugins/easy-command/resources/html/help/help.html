快捷指令插件帮助
· 本插件提供了一些方便操作的快捷指令。
· 本插件的指令除了输入 物品名称 外，还可输入以下 物品类别 进行操作。统称【物品】。
武器、护具、法宝、装备、恢复药、修为药、气血药、丹药、功法、技能书、道具、全部物品
快捷指令
#快捷出售 / 购买【物品】
位于凡仙堂可使用，用于批量操作物品，物品数量为最大值。只涉及一种物品时会根据背包容量和背包灵石修改数量。
#小道消息
位于凡仙堂可使用，花费1000灵石购买小道消息。
#快捷恢复、#快捷恢复80%
不指定百分比可查询当前血量。指定百分比则根据背包恢复药情况自动选择搭配方案，并服用恢复药，恢复血量不会超过给出的百分比。
#快捷服用修为药 / 气血药
服用背包所有的修为药或气血药。
#快捷学习功法 / 技能
学习背包里所有可以学习的功法或者技能书。功法数达到上限时会自动替换低级功法。
#快捷存 / 取灵石 /【物品】
位于万宝楼可使用。最大存取灵石。批量存取物品。
#快捷赠送【物品】@群友
批量赠送物品，不包括灵石。
